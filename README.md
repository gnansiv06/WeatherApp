# SkyCast - Weather App

**Description:**  
SkyCast is a simple interactive weather application that lets users search for any city and view real-time weather information including temperature, humidity, wind speed, and atmospheric pressure. The background changes dynamically based on the weather conditions. Built with HTML, CSS, and JavaScript using the OpenWeatherMap API.

**Technologies Used:**  
- HTML  
- CSS  
- JavaScript  
- OpenWeatherMap API

**Features:**  
- Search any city worldwide  
- View current temperature, humidity, wind speed, and pressure  
- Dynamic background images based on weather conditions (clear, cloudy, rain)  
- Simple and responsive user interface

**How to Use:**  
1. Open `index.html` in your browser.  
2. Enter a city name in the search box and press Enter.  
3. See the updated weather information and background.
